EXAMPLE NAME: IOT_EXAMPLE_XMC47

EXAMPLE REVISION HISTORY:
V1.0.0 initial version 

OVERVIEW: 
XMC4700/XMC4800 IoT example based on DAVE APPs (DAVE v4).
This example demonstrates the implementation of IoT client(s)
on XMC4700/4800 and describes the setup of an environment which 
allows you to experience the IoT client(s) in action.

DESCRIPTION:
Please see "Getting Started - XMC4700_Relax_IoT_Example.pdf"
chapter "Implementing the application"

REQUIREMENTS:
1) "XMC4700 Relax Kit" or
   "XMC4700 Relax Lite Kit" or 
   "XMC4700 Relax Kit for 5V Shields" or 
   "XMC4800 Relax EtherCAT Kit" 
2) Micro USB Cable 
3) Windows PC/Laptop installed with DAVE v4 (Version 4.1.4 or higher)
4) XAMPP web server installer
5) RJ45 ethernet cable
